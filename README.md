# Lista de Feira 🛒

Aplicativo simples para montar listas de compras de **sacolão** e **açougue** e enviá-las prontas pelo **WhatsApp**.

- Sem cadastro e sem servidor: tudo fica salvo no próprio aparelho (localStorage).
- Funciona offline depois do primeiro acesso (PWA).
- Para instalar no celular: abra a página no navegador e use "Adicionar à tela inicial" / "Instalar aplicativo".

Página: https://cvc339.github.io/lista-de-feira/
